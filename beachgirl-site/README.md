## 🚀 Quick Start

### 1. Clone the official repository
```bash
git clone https://github.com/Oriol7272/beachgirl.pics.git
cd beachgirl.pics
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start development server
```bash
npm run dev
```

### 4. Deploy to Vercel
```bash
npm run deploy
```
